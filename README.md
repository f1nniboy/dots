# dots
*my personal dotfiles*

## overview
**window manager**: [dwl](https://github.com/f1nniboy/dwl)
**terminal**: [foot](https://codeberg.org/dnkl/foot)
**bar**: [dwlb](https://github.com/kolunmi/dwlb)
**status**: [someblocks](https://github.com/f1nniboy/someblocks)
**screenshot**: [grim](https://sr.ht/~emersion/grim) + [slurp](https://github.com/emersion/slurp) + [satty](https://github.com/gabm/Satty)
**image viewer**: [imv](https://sr.ht/~exec64/imv)
**launcher**: [tofi](https://github.com/philj56/tofi)
**pdf reader**: [zathura](https://github.com/pwmt/zathura)
**color scheme**: [catppuccin](https://catppuccin.com)
